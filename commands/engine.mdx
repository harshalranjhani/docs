---
title: 'engine'
description: 'Reference for the `engine` command in Genie, the AI-powered CLI companion.'
---

The `engine` command is used to list the current engine being used by Genie. It will print the current engine being used by Genie, which can be either `gemini-1.5-pro` or `gpt-4o-mini`.

## Usage

```bash
genie engine
```

There are no additional arguments required for this command.
