The `chat` command provides a conversational interface where you can interact with Genie in real-time. This command is available exclusively to supporters of the Genie project.


## Usage

```bash
genie chat
```

## Flags

`--safe`: Ensures that the conversation remains appropriate and within safe boundaries.

## Key Features

- **Conversational Interface:** Allows for real-time interaction with Genie, enabling more dynamic and personalized assistance.

- **AI-Powered Responses:** Get instant feedback or help with your queries in a chat-like environment.

```bash
genie chat
```
