---
title: 'reset'
description: 'Reference for the `reset` command in Genie, the AI-powered CLI companion.'
---

The `reset` command allows you to reset your API keys and any of the configurations you have set in Genie. This is useful when you want to start fresh or update your API keys for different services.

## Usage

```bash
genie reset
```

Once you run this command, Genie will prompt you a set of fields to confirm the reset. You can choose to reset your API keys, configurations, or both.

There are no additional arguments required for this command.
