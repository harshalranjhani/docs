title: 'switch'
description: 'Reference for the `switch` command in Genie, the AI-powered CLI companion.'
---

The `switch` command is used to change the underlying AI model used by Genie. This command allows you to switch between different models based on your requirements, such as performance, accuracy, or resource constraints.

## Usage

```bash
genie switch
```

There are no additional arguments required for this command. It will automatically alternate between available the `gemini-1.5-pro` and `gpt-4o-mini` models.
